## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/konytech/ethereum-ercs-demo.git
   ```
3. Install NPM packages
   ```sh
   npm i
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
